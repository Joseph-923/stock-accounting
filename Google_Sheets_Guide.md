# Google Sheets 股票記帳設定指南

## 表格 1：交易記錄（命名為「交易」）

建立這些欄位（A1 開始）：

| A | B | C | D | E | F | G | H |
|---|---|---|---|---|---|---|---|
| 日期 | 股票代號 | 股票名稱 | 交易類型 | 股數 | 成交價 | 手續費 | 總金額 |

### 第一列（標題）範例：
```
日期 | 股票代號 | 股票名稱 | 交易類型 | 股數 | 成交價 | 手續費 | 總金額
```

### 公式（H 欄自動計算）：
```
H2: =IF(D2="買入", E2*F2+G2, E2*F2-G2)
```
向下複製此公式

### 資料輸入範例：
```
2024-04-20 | 2330 | 台積電 | 買入 | 10 | 950 | 20 | （自動計算）
2024-04-21 | 1101 | 台泥 | 買入 | 5 | 45 | 5 | （自動計算）
2024-04-22 | 2330 | 台積電 | 賣出 | 5 | 960 | 10 | （自動計算）
```

## 表格 2：持倉總覽（命名為「持倉」）

建立這些欄位（A1 開始）：

| A | B | C | D | E | F | G | H | I |
|---|---|---|---|---|---|---|---|---|
| 股票代號 | 股票名稱 | 目前股數 | 平均成本 | 總成本 | 目前價格 | 市值 | 未實現損益 | 報酬率 % |

### 步驟 1：手動建立股票清單
在 A2、B2 輸入你持有的股票（例如：2330、台積電）

### 步驟 2：計算目前股數（C 欄）
```
C2: =SUMIFS(交易!E:E, 交易!B:B, A2, 交易!D:D, "買入") - SUMIFS(交易!E:E, 交易!B:B, A2, 交易!D:D, "賣出")
```

### 步驟 3：計算平均成本（D 欄）
```
D2: =IF(C2=0, 0, SUMIFS(交易!E:E * 交易!F:F, 交易!B:B, A2, 交易!D:D, "買入") / SUMIFS(交易!E:E, 交易!B:B, A2, 交易!D:D, "買入"))
```

⚠️ **簡化版本**（如果上面的公式太複雜）：
```
D2: 手動輸入平均成本
```

### 步驟 4：計算總成本（E 欄）
```
E2: =C2 * D2
```

### 步驟 5：輸入目前價格（F 欄）
```
F2: 手動輸入或定期更新 Yahoo 股價
```

### 步驟 6：計算市值（G 欄）
```
G2: =C2 * F2
```

### 步驟 7：計算未實現損益（H 欄）
```
H2: =G2 - E2
```

### 步驟 8：計算報酬率（I 欄）
```
I2: =IF(E2=0, 0, H2/E2*100)
```

## 表格 3：已實現損益（命名為「已實現」）

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| 交易日期 | 股票代號 | 股票名稱 | 買入價 | 賣出價 | 股數 | 損益 |

### 步驟 1：新增賣出記錄
每次賣出時，手動填入買入價、賣出價、股數

### 步驟 2：計算損益（G 欄）
```
G2: =(F2-E2)*D2
```

### 總已實現損益（某個空白格）：
```
=SUM(G:G)
```

## 儀表板（命名為「儀表板」）

在一個獨立表格做總結：

```
A1: 投資總額
B1: =SUM(持倉!E:E)

A2: 目前市值
B2: =SUM(持倉!G:G)

A3: 未實現損益
B3: =SUM(持倉!H:H)

A4: 已實現損益
B4: =SUM(已實現!G:G)

A5: 總損益
B5: =B3+B4

A6: 報酬率 %
B6: =IF(B1=0, 0, B5/B1*100)
```

## 使用流程

### 每次買賣時：
1. 在「交易」表新增一列
2. 填入：日期、代號、名稱、買入/賣出、股數、價格、手續費
3. 自動計算

### 定期（每週或每月）：
1. 在「持倉」表更新「目前價格」(F 欄)
2. 自動重新計算市值、損益、報酬率
3. 查看「儀表板」了解整體狀況

### 賣出時：
1. 在「交易」表新增賣出記錄
2. 也在「已實現」表記錄這筆交易的買入價/賣出價
3. 自動計算已實現損益

## 💡 進階技巧（可選）

### 自動抓股價
在 Google Sheets 用這個公式自動抓台股股價：

```
=GOOGLEFINANCE("TPE:2330","price")
```

在「持倉」表 F2 試試看（把 2330 改成你的股票代號）

## 常見問題

**Q: 公式太複雜怎麼辦？**
A: 先用簡單版本（手動輸入），習慣後再加公式

**Q: 股數不對怎麼辦？**
A: 檢查「交易」表的「交易類型」是否正確填入「買入」或「賣出」

**Q: 可以分享給朋友看嗎？**
A: 可以，Google Sheets 可以設定共享權限
