# 股票記帳系統

一個簡單的台股記帳軟體，幫助你追蹤投資組合、計算損益。

## 功能

- 📊 **儀表板** - 即時查看投資組合、市值、損益
- ➕ **新增交易** - 記錄買入/賣出交易
- 💰 **股利記錄** - 記錄現金股利和股票股利
- 📝 **交易記錄** - 查看和刪除交易
- 📉 **已實現損益** - 計算已賣出股票的損益

## 安裝

1. 安裝 Python 3.8+

2. 安裝依賴：
```bash
pip install -r requirements.txt
```

## 執行

啟動應用程式：
```bash
streamlit run app.py
```

然後在瀏覽器中開啟 `http://localhost:8501`

## 使用方式

1. **新增交易** - 輸入股票代號、名稱、數量、價格
2. **查看儀表板** - 即時查看持倉和報酬率
3. **記錄股利** - 記錄收到的股利
4. **查看損益** - 檢視已實現和未實現損益

## 檔案結構

- `app.py` - Streamlit 前端界面
- `models.py` - 資料庫模型定義
- `database.py` - 資料庫連線和初始化
- `stock_service.py` - 股票業務邏輯
- `requirements.txt` - 依賴列表

## 資料儲存

資料自動儲存在本地的 SQLite 資料庫 `stock_accounting.db`
